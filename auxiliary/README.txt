This folder intended for tools that are outside the code directly,
but go along with this research.
The code may or may not use them.

negwgt - Used in investigation of transport through negative cross
sections by using negative particle weights.  Separate from the code.

quad - Contains an executable and related files that create gauss-legendre
quadrature.  Used by Sn portion of code.

FEDiffSn.inp - Input file for Finite Element Diffusion and Sn portion of code.
